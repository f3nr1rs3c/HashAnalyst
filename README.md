# 🔐 HashAnalyst - Hash Identification Tool

![image](https://github.com/user-attachments/assets/92600c2f-321a-42f6-9c9d-ef69a992bac1)


🔎 **HashAnalyst**, verilen bir hash değerinin **hangi algoritma ile oluşturulduğunu tespit etmek** için geliştirilmiş Python tabanlı bir analiz aracıdır.

Pentest, CTF ve siber güvenlik araştırmalarında **hash analizi** yapmak için kullanılabilir.

---

# 🚀 Özellikler

- 🔍 Hash algoritması tespiti
- ⚡ Hızlı regex tabanlı analiz
- 🎨 Terminal banner (pyfiglet)
- 🌈 Renkli terminal çıktısı
- 🐍 Python tabanlı
- 💻 Linux ve Windows uyumlu

---

# 🛠️ Desteklenen Hash Türleri

Tool aşağıdaki hash algoritmalarını tanıyabilir:

- MD5
- SHA1
- SHA224
- SHA256
- SHA384
- SHA512
- SHA3-224
- SHA3-256
- SHA3-384
- SHA3-512
- Blake2b
- Blake2s
- RIPEMD160
